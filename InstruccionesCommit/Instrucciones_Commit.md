  <h2>Commit - Obteniendo credenciales de acceso para usar la API de Stripe desde Laravel</h2>
  <ol>

      <br>
      <em>
        *Ahí se encuentran la <code>Clave publicable</code> y la <code>Clave secreta</code> que se usan en
